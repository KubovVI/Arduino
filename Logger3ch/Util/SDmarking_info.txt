Программа SDmarking.exe предназначена для разметки SD-Flash карточек для последующей записи данных на микропроцессорной плате.
Программа созает один большой файл, содержащий требуемое число блоков по 512 байт.
Первый блок содержит текстовый маркер, по которому микропроцессор может обнаружить начало файла. 
Максимальная длина маркера не более 16 символов.
Вслед за маркером, со смещением 16 байт от начала блока идет 4 байтный размер файла в 512-байтных блоках.
После первого блока идут пустые блоки, заполненные шестнадатиричными кодами 0xFF.

Программа получает необходимые параметры из файла SDmarking.ini.
Здесь можно задать:
- имя файла - fileName;
- текстовый маркер - marker;
- размер файла в блоках 512-байт - blocks. 
Фактический размер файла будет на один блок больше указанного в параметре blocks за счет блока с маркером.

Каждый параметр указывается с новой строки. 
Значения параметра связывается с его именем посредством символа "=".
Строки начинающиеся с символа ";" являются комментариями, и не обрабатываются программой.

Пример файла инициализации.
; SD-card marking initialisation
fileName=Logger.bin
marker=Kubov V.I. 2014
blocks=200000

При запуске программа просит указать путь к файлу записи с помощью стандартного окна файлового менеджера. 
Следует указать путь к накопителю с SD-Flash картой.
После запуска программа осуществляет разметку файла с записью пустых блоков на карточку.
Для больших размеров файла запись может длиться несколько десятков секунд.
Следует подождать завершения разметки.
О завершении разметки свидетельствует всплывающее окно с указанием файла, маркера и размеров файла. 

Последовательность действий по подготовке SD-Flash карты:

1. Отформатировать карточку под FAT16 или просто FAT.
Микропроцессор будет писать данные на карту блок за блоком подряд, не используя файловую систему.
Поэтому память для файла должна выделяться одним сплошным куском. 
Форматирование гарантирует, что память на карточке будет выделена одним сплошным куском. 

2. Изменить файл SDmarking.ini в соответствии со своими требованиями.
2.1. Настроить маркер под программу микропроцессора, которая именно его и будет искать.
Маркер должен быть достаточно уникальным, чтобы имя файла или информация в файле не были спутаны с маркером.
2.2. Указать размер файла в блоках 512-байт. 
Программа микропроцессора будет его учитывать, чтобы не выйти за границы файла.
Размеру 200000 блоков соответствует, примерно, 100 мегабайт файл. Это достаточно для карточки на 128Mb. 
2.3. Программа микропроцессора не анализирует имя файла. Его можно задать любым, но не совпадающим с маркером. 

3. Выполнить разметку карточки программой SDmarking.exe. 
3.1. Запустить программу.
3.2. Дождаться ее завершения.

4. Извлечь размеченную карточку.
